# Query
[![npm version](https://badge.fury.io/js/query.svg)](https://badge.fury.io/js/query)

A composable for handling query parameters for Vue and Inertia.

## Installation
You can install the package via npm:

```bash
npm i query
```

## Usage

```javascript
import * from 'query';
```
